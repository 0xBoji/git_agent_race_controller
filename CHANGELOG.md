# Changelog

All notable changes to this project will be documented in this file.

## [0.1.2] - 2026-04-07

### Added
- `camp_status` in `garc status` output.
- `camp_update_exit_code` and `camp_update_stderr` in checkout JSON.
- CAMP REST bridge fallback for peer discovery when available.

### Changed
- `garc up --json` now flushes its preamble before delegating to `camp up`.
